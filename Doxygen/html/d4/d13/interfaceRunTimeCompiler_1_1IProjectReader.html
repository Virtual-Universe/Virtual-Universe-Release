Detailed Description

This interface defines a way of working with project-reader. Currently the only project-reader developed is the one for c# projects, but many other readers can be developed for various languages/IDEs.

Member Function Documentation

bool RunTimeCompiler.IProjectReader.CanOpen ( string  filename)

This method will be used to check if the specified project file can be opened. Implementations may involve operations like: checking the file extension, reading the file header (or some specific sectors) and getting the file version. Note! If the call returns true it does not guaranties calls to ReadProject will be successfull.

Parameters
filenameThe name of the project file. It should include path (absolute or relative).
Returns
True if the project-reader "believes" he can read the project file.
